

# Mohammad Junaid — Premium Portfolio Website

## Overview
A modern, dark-mode single-page portfolio website with glassmorphism design, animated gradient backgrounds, and smooth scroll animations. The site will showcase Mohammad Junaid as a Backend & Full-Stack Software Engineer.

---

## Design System
- **Theme**: Dark-mode first with animated gradient background (deep blue → purple → neon cyan)
- **Style**: Glassmorphism cards with blur and subtle transparency
- **Typography**: Inter font, large bold headings, clean letter-spacing
- **Animations**: Fade-in on scroll, hover glow effects on cards, smooth page-load animations
- **Profile Photo**: Your uploaded photo (`IMG_8915.jpg`) with a stylish rounded glass effect

---

## Sections

### 1. Sticky Navbar
- Minimal transparent navbar with smooth scroll links to each section
- Name/logo on the left, navigation links on the right
- Becomes slightly opaque on scroll

### 2. Hero Section

- Subtext highlighting your core tech stack

## Backend (Lovable Cloud)
- A Supabase edge function to handle contact form submissions
- Integrates with **Resend** API to deliver emails to your inbox
- You'll need to provide a Resend API key and your receiving email address during implementation

## Responsive & Performance
- Fully responsive from mobile to desktop
- Lightweight CSS animations (no heavy libraries)
